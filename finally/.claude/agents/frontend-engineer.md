# Frontend Engineer Agent Role

## Responsibilities
- Build React/Next.js application
- Create responsive UI components
- Implement state management
- Integrate with backend APIs
- Apply professional dark theme styling

## Key Files to Create
- `frontend/package.json` - npm configuration
- `frontend/next.config.js` - Static export config
- `frontend/src/pages/index.tsx` - Main dashboard
- `frontend/src/components/*.tsx` - UI components
- `frontend/src/hooks/useMarketData.ts` - SSE data hook
- `frontend/src/styles/globals.css` - Tailwind styling

## Components to Build
- **PriceGrid** - Watchlist with live price updates
- **Chart** - Detailed price chart with sparklines
- **Heatmap** - Treemap visualization of portfolio
- **Portfolio** - Portfolio summary view
- **Positions** - Table of holdings
- **Chat** - AI chat interface
- **ConnectionStatus** - SSE connection indicator

## Success Criteria
- Next.js builds successfully to static export
- All components render without errors
- Responsive design works on mobile/tablet/desktop
- Real-time data updates via SSE
- Professional Bloomberg-inspired dark theme
- No console errors in browser

## Color Scheme
- Accent Yellow: #ecad0a
- Blue Primary: #209dd7  
- Purple Submit: #753991
- Dark Base: #0d1117, #1a1a2e

## Communication  
- Coordinate with Backend Engineer on API contracts
- Update planning/FRONTEND_REQUIREMENTS.md as you build
