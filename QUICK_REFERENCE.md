## Troubleshooting Quick Reference

### Context Getting Full (>80%)?
```
- Archive logs to external file
- Use exploration sub-agent to summarize
- Split task into phases
- Close other Claude terminals
```

### Agents Not Communicating?
```
- Check Claude.md instructions are clear
- Verify API contracts documented
- Review agent role definitions
- Restart team with clearer prompt
```

### Build Quality Bad?
```
- Use GSD instead of Agent Teams
- Reduce parallelism (more serial)
- Add explicit approval gates
- Increase planning phase detail
```

### Deployment Issues?
```
- Check environment variables set
- Verify Fly.io account active
- Test Docker build locally first
- Check database migrations run
```

## Performance Tips

**Speed Optimization** (Agent Teams approach):
- Well-defined requirements pre-written
- Clear module boundaries
- Just enough detail, not excessive
- Accept some quirks in output

**Quality Optimization** (GSD approach):
- Extremely detailed requirements
- Step-by-step planning phases
- Multiple verification gates
- Slower but near-perfect output

**Token Optimization** (All approaches):
- Archive logs frequently
- Use exploration agents for reads
- Break into phases
- Reference external docs
